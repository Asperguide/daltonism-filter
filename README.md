# daltonism-filter
This is a website attempting to apply a daltonisme filter to help these people to read the page with less difficulty than on other websites.


## Dislaimer
This project is going to be archived because it has achieved it's core goal, provide an easy all in one way of embedding daltonism filters into a page, you can find the implementation versions in the webpage version and in the tampemonkey folder. The code is pure javascript manipulating front-end css and html, it is documented via the comments in the code, and as long as the HTML5 norm stands, same for CSS3, it should work. The code works as a drop in for any website that wishes to use it. The only thing I ask is for you to credit me.
